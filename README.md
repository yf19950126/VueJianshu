# 我的简书

> 仿简书,使用nuxt+Vue+node完成简书所有功能

## 安装和打包

``` bash
# 安装依赖
$ npm install # Or yarn install

# 启动服务器,使用localhost:3000打开调试
$ npm run dev

# 打包、上线部署
$ npm start
```
